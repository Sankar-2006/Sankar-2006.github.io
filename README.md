# Uma Sankar Portfolio

Professional dark-themed portfolio for an ECE student focused on embedded systems.

## Structure

- index.html
- assets/css/main.css
- assets/js/data.js
- assets/js/app.js

## Edit Your Data (Single File)

Update all your portfolio content in:

- assets/js/data.js

This includes:

- Name, role, about text
- Email, phone, location
- GitHub, LinkedIn, resume links
- Skills, projects, education, certifications
- Loader style option (`loaderType: "name"` or `loaderType: "normal"`)

## Features

- Dark professional UI
- Hidden scrollbar
- Responsive layout for desktop/tablet/mobile
- Advanced reveal and hover animations
- Desktop-only custom cursor animation
- Configurable loading animation (name loader or normal pulse loader)
- Contact form that opens mail app directly via `mailto`
